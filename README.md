# ITZFIZZ — Scroll-Driven Hero Section Animation

A scroll-driven hero section built as an assignment for the **Itzfizz Digital Web Development Internship**.

## 🔗 Live Demo
[View Live Site](https://Kushmarai.github.io/itzfizz-hero)

## ✨ Features
- Animated loader with percentage counter
- Staggered character-by-character headline reveal
- Scroll-driven animations tied to scroll progress (not time-based)
- Parallax layers — code window, orbit rings, and floating tech tags all move at different speeds
- Animated stat counters on load
- Custom magnetic cursor with ring lag effect
- Scroll progress bar
- Continuous marquee tech-stack ticker
- Scroll-triggered reveals for roles, perks, and CTA sections

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP + ScrollTrigger
- Tailwind CSS (CDN)
